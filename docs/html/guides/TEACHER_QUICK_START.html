Getting Started (5 Minutes)

1. Register

Go to: liphant.co/register - Select “I’m a Teacher” - Enter your details - Verify your email

2. Complete Your Profile

Go to: liphant.co/teacher/profile

Must Complete: | Field | Why It Matters | |——-|—————-| | Bio | Parents want to know you | | Specializations | Match with right families | | Hourly Rate | Set your session fee | | Location | Parents find you nearby | | Photo | Build trust instantly |

3. Set Availability

Go to: liphant.co/teacher/availability - Add your weekly time slots - Parents can only book when you’re available

4. Get Verified

Our team will review your profile. Once approved, you’ll receive a Verified Badge and start receiving bookings!

How Bookings Work

Parent Requests → You Accept → Session Happens → Get Paid + Review

Manage bookings at: liphant.co/teacher/bookings

Your Dashboard
